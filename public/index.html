Run this yourself or wire it into Claude Code

            
Same code as this public instance. Three ways to get it.


            

                
🤖 Use in Claude Code (MCP)

                
One line, defaults to this hosted backend. Restart Claude Code after install.

                

                    
                    
claude mcp add opengraph -- npx -y @krytonlabs/opengraph-mcp

                

            


            

                
🐳 Self-host with Docker

                
Run your own instance. Edit env vars to taste — see .env.example.

                

                    
                    
docker run -d --name opengraph -p 6736:6736 \
  -e PUBLIC_URL=https://og.yourdomain.com \
  ghcr.io/kmanan/opengraph:latest

                

                
Prefer a UI? Add the stack from docker-compose.yml via Portainer and fill in env vars on the form.

            


            

                
⚙️ Self-host both, MCP points at your backend

                

                    
                    
claude mcp add opengraph -- npx -y @krytonlabs/opengraph-mcp \
  --env OPENGRAPH_BACKEND=http://localhost:6736
